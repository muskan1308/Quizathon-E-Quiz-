The project provides following special functionalities::

1.Matching of password and confirm password.
2.When Admin deletes any question from the middle, the Qno count is reordered and Auto Incremented.
3.If login with admin username and password then you are directed to the admin panel.
4.The result is showed for each subject.
5.Admin can delete any question, or add new questions.
6.Admin can update,delete user.
7.Admin can add another admin.
8.When user clicks on disagree, his session is terminated.
9.The project uses session management.
10. There is a checkpoint class that checks if the user has agreed to the terms and conditions or not.
11. Clicking on logout will also terminate the session.
